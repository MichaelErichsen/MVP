1.2Adding TMG data

If you want to add the underlying TMG H2 database it is an effort in itself. However I have already embedded the H2 engine in the project for you.

Create database and tables

You still need an H2 installation on your system.

Download the H2 zip file from

http://www.h2database.com/html/download.html

And run it. That should create a shortcut to the H2 console, from where you can run the DDL script:

It is located in

•Project org.historyresearchenvironment.tmg.h2

•Package org.historyresearchenvironment.tmg.h2.Dbf2H2

•DDL script CreateTmg9.sql

You probably need to remove the first ALTER and DROP statements.

Creating from inside Eclipse

If you want to do it from inside Eclipse then first install DTP (Data Tools Platform):

Help  Install new software

Using site http://download.eclipse.org/datatools/1.14.0.201701131441/repository/

After installation you can open the Database Development perspective and define a Generic JDBC connection.

Eclipse DTP H2 Tutorial:

http://theopentutorials.com/tutorials/eclipse/dtp/eclipse-dtp-configure-h2-datasource-using-data-source-explorer/

using your downloaded H2 package, in my case h2-1.4.196.jar, database TMG9 and userid sa, no password, and URL jdbc:h2:~/TMG9
