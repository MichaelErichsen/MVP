Development Toolbox Installation

Java

Install Java JDK from

http://www.oracle.com/technetwork/java/javase/downloads/index.html

Please use Java 1.8 or higher. I am running Java 8, level 144 64 bit.

Eclipse

Install Eclipse Oxygen Windows 64 bit version 4.7 for RCP Developers, eclipse-rcp-oxygen-R-win32-x86_64.zip, from

http://download.eclipse.org/technology/epp/downloads/release/oxygen/R/eclipse-rcp-oxygen-R-win32-x86_64.zip

Unpack wherever you like. I have put it in a folder under Program Files/HRE, running Windows.

Install new software:

Use the Eclipse update manager to install the "Graphical Editing Framework Zest Visualization Toolkit". You may have to unflag Group items by category to see Eclipse Zest.

The GEF component is separated into three plug-ins:

Draw2d (org.eclipse.draw2d) - the lightweight toolkit for painting and layout on an SWT Canvas

GEF (org.eclipse.gef) - an interactive MVC framework built on top of Draw2d

Zest (org.eclipse.zest) - a visualization toolkit built on top of Draw2d
