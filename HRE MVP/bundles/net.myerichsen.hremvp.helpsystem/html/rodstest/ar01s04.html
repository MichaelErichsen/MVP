Help

A versatile system is provided within the HRE application to assist the user with additional information about operation of the software.   Access to the Help system is available from the Help item in the main menu, which then provides an indexed search facility

From any location in the application, access to the relevant section of the Help system may also be achieved by use of the Help Icon (symbol) on the window, or by use of the F1 key.   These will directly access the that section of the Help material.  

A Wiki is also provided, and this includes a more extensive information set about the software
