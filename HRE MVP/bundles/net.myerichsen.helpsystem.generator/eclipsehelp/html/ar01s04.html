Defining the   Data Model

Define the logical data model as the data items that the feature will expose to the user, and which supporting data is needed.

Define the physical data model as the tables, view and columns in the HRE data base that should be used by the feature.   SQL Power Architect can be used as a tool for this. The Community Edition is available at   http://www.bestofbi.com/page/architect_download_os .
